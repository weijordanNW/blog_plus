**一、基础概念**
---
首先，我们需要了解一下 Flex 布局中的一些概念和术语：

flex 容器（flex container）

flex 项目（flex item）

主轴（main axis)

交叉轴（cross axis）

占用主轴空间（main size）

占用交叉轴空间（cross size）

线轴起止点（main start、main end、cross start、cross end）

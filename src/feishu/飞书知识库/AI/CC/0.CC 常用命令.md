## 常用命令速查
### 启动Claude Code
```plaintext
claude
```

### 查看上下文占用
```plaintext
/context
```

### 压缩上下文
```plaintext
/compact
```

### 清空上下文
```plaintext
/clear
```

### 恢复历史会话
```plaintext
/resume
```

### 查看MCP 列表
```plaintext
/mcp
```

### 📝 ContextMenuMgr (Context Menu Manager - 右键菜单管理器)
- 项目地址：[https://github.com/PLFJY/ContextMenuMgr](https://github.com/PLFJY/ContextMenuMgr)

- 功能介绍：专为 Windows 系统设计的右键菜单管理工具。它能启用或禁用文件、文件夹、桌面、回收站等多种场景下的右键菜单项，并清理软件卸载后留下的残留项，让右键菜单回归简洁。工具通过隐藏而非删除注册表项的方式进行操作，以避免误操作导致系统问题。

### 💐 Floral-Notepaper (花笺)
- 项目地址：[https://github.com/Achilng/floral-notepaper](https://github.com/Achilng/floral-notepaper)

- 功能介绍：一款基于 Tauri 2 + React 构建的轻量、优雅、现代化的本地便签工具。它支持 Markdown 编辑与实时预览，可通过全局快捷键（默认 `Ctrl+Space`）随时唤出，并支持将笔记以“磁贴模式”固定在桌面上以供快速查阅与复制。此外，它还支持 `.md` 文件的导入与导出。

### 🧹 BleachBit
- 项目地址：[https://github.com/bleachbit/bleachbit](https://github.com/bleachbit/bleachbit)

- 功能介绍：一款跨平台（Windows、Linux、macOS）的系统清理工具，旨在释放磁盘空间并维护用户隐私。它可以清理系统缓存、临时文件、浏览器历史记录、Cookie 以及损坏的快捷方式等。同时，它也支持为 Firefox、Chrome、Java 等多种第三方应用清理垃圾文件。

### 📚 ReadAny
- 项目地址：[https://github.com/codedogQBY/ReadAny](https://github.com/codedogQBY/ReadAny)

- 功能介绍：一款 AI 驱动的开源跨平台电子书阅读器。它通过将大语言模型与 RAG（检索增强生成）技术结合，支持用户与书籍进行智能对话，并能基于书籍内容进行超出关键词的语义搜索。此外，它还拥有丰富的笔记与高亮管理系统、文本转语音（TTS）、阅读统计、WebDAV 多端同步以及 Markdown、Obsidian 等多种笔记导出格式。

### 📺 TabulaBili (初见哔哩增强版)
- 项目地址：[https://github.com/tjsky/TabulaBili](https://github.com/tjsky/TabulaBili)

- 功能介绍：一款用于解决 B 站首页“信息茧房”问题的 Chrome 浏览器扩展。它利用 Chrome 的底层网络拦截机制，在拉取首页推荐流时精准剥离请求中的身份信息 Cookie，让 B 站算法将你识别为“新访客”，从而返回更广泛的热门内容流。此过程不影响你的登录状态，因此你依然可以观看 1080P+ 高清画质、发送弹幕、评论和点赞。

### 📈 Stock-SDK
- 项目地址：[https://github.com/chengzuopeng/stock-sdk](https://github.com/chengzuopeng/stock-sdk)

- 功能介绍：一个为前端和 Node.js 环境设计的零依赖、轻量级 JavaScript SDK，旨在让开发者用熟悉的 JavaScript/TypeScript 优雅地获取股票行情数据。它支持在浏览器或 Node.js 中直接获取 A 股、港股、美股、公募基金的实时行情、历史 K 线、分时走势以及技术指标（如 MA、MACD、KDJ 等）数据。此外，它还提供资金流向、龙虎榜、大宗交易、融资融券等扩展数据接口，并原生支持 TypeScript。

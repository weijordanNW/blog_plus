## CloakBrowser-隐身浏览器
**GitHub 地址**：[https://github.com/CloakHQ/CloakBrowser](https://github.com/CloakHQ/CloakBrowser)
### 项目简介
CloakBrowser 是一款基于 Chromium 深度改造的隐身浏览器，通过 **C++ 源码级补丁** 实现指纹隐藏，可轻松通过各类机器人检测系统，完美兼容 Playwright/Puppeteer 自动化框架。
### 核心特性
- 49 项源码级指纹补丁，覆盖画布、WebGL、音频、字体、GPU、WebRTC 等维度

- 支持 `humanize=True` 实现真人级鼠标、键盘、滚动行为模拟

- reCAPTCHA v3 评分可达 0.9（人类水平），完美通过 Cloudflare Turnstile

- 原生支持 SOCKS5 代理，自动根据代理 IP 匹配时区 / 区域

- 提供浏览器配置管理器，可替代 Multilogin/GoLogin/AdsPower

- 免费开源，无订阅、无使用限制，自动更新二进制文件

### 快速使用
bash

运行
```plaintext
# Python 安装
pip install cloakbrowser

# JavaScript 安装npm install cloakbrowser playwright-core
```

## Bun-极速 JavaScript 运行时
**GitHub 地址**：[https://github.com/oven-sh/bun](https://github.com/oven-sh/bun)
### 项目简介
一站式 JavaScript/TypeScript 开发工具包，包含**高速 JavaScript 运行时、包管理器、打包器、测试工具**，可完全替代 Node.js，启动速度与内存占用大幅优化。
### 核心特性
- 基于 JavaScriptCore 引擎，启动速度极快，内存占用低

- 内置 TS/JSX 支持，无需额外配置

- 兼容 Node.js API，可无缝迁移现有项目

- 集成包管理器（比 npm/yarn 快 10 倍 +）、测试运行器、HTTP 服务

- 支持 Linux/macOS/Windows 全平台，持续活跃更新

### 快速使用
bash

运行
```plaintext
# macOS/Linuxcurl -fsSL https://bun.com/install | bash# Windows
powershell -c "irm bun.sh/install.ps1 | iex"
```

## RuView - WiFi 无感感知系统
**GitHub 地址**：[https://github.com/ruvnet/RuView](https://github.com/ruvnet/RuView)
### 项目简介
基于 WiFi 信号的**无摄像头人体感知平台**，利用 ESP32 采集 CSI 数据，实现穿墙人体姿态估计、生命体征监测、存在检测，无隐私泄露风险。
### 核心特性
- 无需摄像头、无需穿戴设备，穿墙检测人体

- 实时监测呼吸频率、心率、睡眠质量、摔倒报警

- 支持多人计数、动作识别、环境映射

- 基于 ESP32-S3（约 9 美元）低成本硬件，本地边缘计算

- 支持 3D 点云融合、多节点 Mesh 组网，隐私安全

### 快速使用
bash

运行
```plaintext
# Docker 模拟体验docker pull ruvnet/wifi-densepose:latest
docker run -p 3000:3000 ruvnet/wifi-densepose:latest
```
